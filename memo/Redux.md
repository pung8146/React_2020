## Props vs State ##
데이터를 다룰 때 사용되는 개념이다.


### Props ###
부모 컴포넌트가 자식 컴포넌트한테 전달 하는 데이터로,(자식 입장에서) 읽기 전용이다
<Child value="value" />

### State ###
1. 자신이 들고 있는 값을 말한다( 읽기 전용인 Props와 비교해보자면,쓰기 전용이라고 볼 수 있겠다.)

React 는 

페이스북에서 개발한 웹 라이브러리 리액트는 앵귤러와 달리
UI만 제공합니다.

순수함수 는 입력값이 출력값이 같다
(랜덤 함수사용X , 외부 상태 변경)

State.불변 변수로 관리
(복잡도랑 버그가 발생활 가능성이 낮아짐)

가상돔을 통해 UI에 적용해서 #실제적용된#것만 적용되어줌
