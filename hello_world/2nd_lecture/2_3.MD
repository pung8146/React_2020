## 2.3.1 매개변수에 추가된 기능 ##

** 매개변수 기본값 ** : ES6부터 매개변수에 기본값을 줄 수 있다

코드 2 46 매개변수에 기본값 주기
```
function printLog(a = 1) {
    console.log({ a });
}
printLog(); // { a: 1 }
```
인수 없이 함수를 호출하므로 a에는 undefined가 입력된다.
